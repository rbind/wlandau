Dynamic Ledger

Personal accounting can be frustrating when there are a lot delayed transactions. With pending charges and charges that take some time to even reach the bank, it can be hard to know how much you actually have left. That's why I created Dynamic Ledger. Unlike most other accounting programs, it gives you control over your finances even when you have several delayed transactions. With it, you can be exactly as frugal as you need to be, and you can easily avoid spending more than you actually have.

This is not my personal ledger, of course, but it is a useful example. If the file above is called ledger.txt, then I can open Terminal (or another command line interface program) and type 
dl ledger.txt
And I get a summary of my ledger. The summary tells me that I have $0.00 "available" on my credit card and $1,335.00 "available" in my credit card. These are the balances that I should see on my banks' websites and mint.com if I've done my bookkeeping correctly. But in reality, I owe my credit card company $45.50 and I only have $1,289.50 of real money in my bank account because of the two delayed transactions at the top of the ledger file.

Currently, the program is an open source Linux/Unix-style package written in C. You can clone it from my GitHub page or download the source as a tarball from here. It should be compatible with all platforms with a C compiler. Installation and usage instructions are explained in the man page and pdf tutorial.

I've done my personal accounting this way for years, and it's given me a lot of peace of mind. It's gratifying to be able to finally package up this system and distribute it as a program that other people can use.

Eventually, I want to turn Dynamic Ledger into an app for Mac, iphone, and Android. I've written most of the backend already. The plan is to learn Objective C and Cocoa and then write a spreadsheet program that not only computes summaries, but also allows the user to condense and clean ledgers (supported in the current version), click and drag around rows and columns, etc. Please let me know what you'd like to see.
